Changelog

This page lists all the biggest changes in the past updates, up to the current version (as of writing this).

Update Notes

  • Version 1.0.9 (July 24th, 2015)
    • Updated documentation to a prettier and more up-to-date version
    • Added support for a completely ad-free version of BlogIt
    • Updated project structure (please replace the files and re-run Update Pods.command)
  • Version 1.0.8 (May 25th, 2015)
    • Updated cocoapod script (please run Update Pods again)
  • Version 1.0.7 (April 20th, 2015)
    • Fixed crashing in XCode 6.3 (please run Update Pods again)
    • Fixed comment popup not working
    • Fixed other small issues with commenting
  • Version 1.0.6 (March 4th, 2015)
    • Fixed crashing on categories that don't have posts or have issues with posts
    • Fixed crashing when sharing pages
    • Fixed a problem with the first category being skipped
  • Version 1.0.5 (February 20th, 2015)
    • Fixed minor issues with AdMob
    • Tested & confirmed that BlogIt works on iOS 7 and above
  • Version 1.0.4 (February 9th, 2015)
    • Fixed a crash with certain blogs
  • Version 1.0.3 (February 5th, 2015)
    • Fixed an issue with CocoaPods/Podfile, where the latest Google AdMob SDK was suddenly removed from CocoaPods, breaking the project
  • Version 1.0.2 (February 3rd, 2015)
    • Added INFINITE SCROLLING! I know many of you have been waiting for this one, and it's finally here. Supports both latests posts and categories and allows you to scroll, well, infinitely through all your posts!
    • Fixed a problem when manually refreshing
    • Added a custom map page
    • Fixed "Update Pods.command"
  • Version 1.0.1 (January 27th, 2015)
    • Category support! Categories are automatically added to the sidebar, but you can exclude certain categories
    • Push Notifications now integrate directly with WordPress (check the Documentation)
    • Much, much better support for both iAd and AdMob
  • Version 1.0.0 (January 19th, 2015)
    • First Release
